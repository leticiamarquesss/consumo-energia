# ⚡ Calculadora de Consumo Elétrico

Um programa simples em Python que calcula o consumo mensal de energia elétrica de um aparelho.

## 🎯 Objetivo

Ajudar usuários a estimar o gasto de energia elétrica com base no uso diário.

## 🧪 Tecnologias

* 🐍 Python
* 💻 VS Code
* 🌐 GitHub

## 📐 Fórmula utilizada

consumoMensal = (potencia * horasDia * 30) / 1000

## 💰 Cálculo de custo

custo = consumoMensal * 0.75

## ▶️ Como executar

1. Abra o terminal
2. Vá até a pasta do projeto
3. Execute:

python app.py

## 📊 Exemplo

Digite o nome do aparelho: Geladeira
Digite a potência: 150
Digite horas por dia: 10

Resultado:
Consumo estimado: 45 kWh/mês
